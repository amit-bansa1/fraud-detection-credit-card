## Dataset
Download the dataset from Kaggle and place in the `data/` folder:
- Link: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- File needed: `creditcard.csv`
- Size: ~144MB (excluded from repo due to GitHub file size limits)# fraud-detection-credit-card
End-to-end credit card fraud detection model using XGBoost.  Handles extreme class imbalance (0.17% fraud rate) via SMOTE  and threshold tuning. Evaluated on Precision, Recall, and  AUC-PR — not accuracy. Python | scikit-learn | XGBoost | imbalanced-learn
